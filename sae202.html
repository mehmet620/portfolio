Web Radio portable sur ESP32

SAÉ 2.02 · Semestre 2 · Binôme · 22h

Conception d'une web radio IoT : un microcontrôleur ESP32 qui lit des flux MP3/AAC depuis internet, contrôlable depuis le téléphone via MQTT. Un projet qui mêle électronique, programmation embarquée et protocoles réseau.

Ce qu'on a fait

  • Câblé et interfacé une carte Adafruit HUZZAH32 ESP32 avec un CoDec audio VS1053 via le protocole SPI
  • Programmé en C++ sous IDE Arduino en adaptant des bibliothèques tierces (Baldram, ESP32_VS1053_Stream)
  • Ajouté le contrôle du volume, des basses, des aigus et de la spatialisation depuis le port série
  • Implémenté WiFiManager pour se connecter à n'importe quel réseau Wi-Fi sans mettre les identifiants en dur
  • Intégré le protocole MQTT (PubSubClient) pour piloter la radio depuis l'app Android IoT MQTT Panel
  • Accédé directement aux registres matériels du VS1053 pour créer des fonctions de spatialisation non disponibles dans les bibliothèques

Concepts mis en œuvre

  • Protocole SPI — bus de communication série entre le microcontrôleur et le CoDec audio
  • MQTT — protocole de messagerie IoT publish/subscribe, idéal pour les objets connectés
  • WiFiManager — portail captif pour configurer le Wi-Fi dynamiquement sans modifier le code
  • Registres matériels — accès bas niveau aux fonctions du VS1053 pour étendre ses capacités
