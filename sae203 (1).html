What we did

  • Set up a Django project with Python and connected it to a MySQL database
  • Designed the data schema and implemented CRUD operations for servers, services, applications and users
  • Built the web interface in HTML/CSS with Django templates and forms
  • Used Pillow for image handling and populate_db to seed the database with initial data
  • Worked as a team of three using Git/GitHub to version the code and split tasks
  • Planned the project with a Gantt chart (18 May – 12 June 2026)
  • Wrote an installation and deployment guide and presented the project with a live demo

How to run the project

1

Install Python from python.org — check "Add Python to PATH" during setup

2

Download and extract the ZIP from GitHub, open the folder in VS Code

3

Open a terminal and navigate: cd Downloads\SAE_R203-main\PythonProject

4

Allow scripts (once): Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

5

Create the virtual environment (once): python -m venv venv

6

Activate it: venv\Scripts\activate

7

Install dependencies (once): pip install django then pip install Pillow

8

Init the database (once): python manage.py migrate then python manage.py populate_db

9

Start the server: python manage.py runserver → open http://127.0.0.1:8000
