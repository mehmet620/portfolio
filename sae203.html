Application web avec base de données

SAÉ 2.03 · Semestre 2 · Groupe de 3 · Compétence RT3

En groupe de 3, on a conçu et déployé un site web hébergé sur une VM Linux pour consulter et manipuler une base de données fournie par l'IUT — première expérience de développement web collaboratif.

Ce qu'on a fait

  • Configuré une machine virtuelle Linux et installé un serveur web + un SGBD (MariaDB)
  • Modélisé le schéma de données fourni et développé les fonctions CRUD (consultation, insertion, modification)
  • Conçu l'interface web en HTML/CSS et connecté les formulaires à la base de données
  • Travaillé à 3 avec Git/GitHub pour versionner le code et se répartir les tâches
  • Rédigé un diagramme de Gantt pour planifier et suivre l'avancement
  • Produit une procédure d'installation et de déploiement (2–5 pages) expliquant nos choix techniques
  • Présenté le projet en 5 minutes avec démo, suivi de 10 minutes de questions

Concepts mis en œuvre

  • Architecture web — relation entre serveur web, SGBD et client navigateur
  • CRUD — Create, Read, Update, Delete : les quatre opérations de base sur une BDD
  • Travail collaboratif avec Git — branches, commits, résolution de conflits
  • Diagramme de Gantt — planifier et répartir les tâches dans le temps

Apprentissages critiques

AC0314

Architecture d'un site web — comprendre comment un serveur web, une base de données et un client s'articulent ensemble.

AC0315

Gestion de données — choisir et configurer un SGBD, concevoir un schéma adapté au besoin métier.

AC0316

Travail collaboratif — utiliser Git en équipe pour la première fois, c'est plus compliqué que prévu mais vraiment utile.

R107/R209

Ressources mobilisées — fondamentaux de la programmation, technologies web, sources et traitement des données.

Bilan réflexif

Ce projet m'a appris que coder en groupe c'est autant une question d'organisation que de technique. Les conflits Git au début nous ont fait perdre du temps, mais on a vite compris l'importance des branches et du découpage clair des tâches. Présenter devant les enseignants m'a aussi appris à défendre mes choix techniques à l'oral.
